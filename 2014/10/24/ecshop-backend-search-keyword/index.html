ecshop后台关键字搜索优化

By chunze.huang

10月 24 2014 Updated:1月 28 2015

Contents

ecshop在商品列表中搜索需要非常精准，如果添加有2个用空格分开的关键字是搜索不到结果的。需要作如下修改才可实现过关键字的搜索：

首先要在文件‘admin/include/lib_goods.php’中修改函数，或者直接搜索
$where .= " AND (goods_sn LIKE '%" . mysql_like_quote($filter['keyword']) . "%' OR goods_name LIKE '%" . mysql_like_quote($filter['keyword']) . "%')";
看看，首先要注释掉上面的语句，然后添加下面的语句：

`
/ 关键字 /
if (!empty($filter[‘keyword’]))
{
$goods_sn_statment = ‘’;
$goods_name_statment = ‘’;
$s_keyword = trim($filter[‘keyword’]);
$s_keywords = explode(‘ ‘, $s_keyword);
array_filter($s_keywords);
if(!empty($s_keywords)){
$v_keyword = ‘’;
foreach($s_keywords as $v_keyword){
$goods_sn_statments[] = “ goods_sn LIKE ‘%” . mysql_like_quote($v_keyword) . “%’”;
$goods_name_statments[] = “ goods_name LIKE ‘%” . mysql_like_quote($v_keyword) . “%’”;
}

            if(!empty($goods_sn_statments)){
                $goods_sn_statment = ' ( '.join(' AND ', $goods_sn_statments).' ) ';
            }
            if(!empty($goods_name_statments)){
                $goods_name_statment = ' ( '.join(' AND ', $goods_name_statments).' ) ';
            }

        }
        $where .= " AND ( ".$goods_sn_statment.' OR '.$goods_name_statment." )";

// $where .= “ AND (goods_sn LIKE ‘%” . mysql_like_quote($filter[‘keyword’]) . “%’ OR goods_name LIKE ‘%” . mysql_like_quote($filter[‘keyword’]) . “%’)”;
}
`
