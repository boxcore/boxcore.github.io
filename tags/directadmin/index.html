设置时间

1. 修改系统时间

2. 修改php时间

编辑如下内容

date.timezone = "Asia/Shanghai"
执行重载service directadmin reload

3. 修改mysql时间

使用root账号执行: SET GLOBAL time_zone = '+08:00';
然后执行select now()查看当前时间
